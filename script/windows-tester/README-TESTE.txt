ERI TESTER - MODO DESENVOLVIMENTO

1. Abra o arquivo ABRIR_TESTE_DEV.bat com dois cliques no Windows.
2. O Windows precisa ter Node.js e Rust instalados para o modo dev funcionar.
3. O projeto pode ser editado normalmente dentro desta pasta de teste.
4. As alteracoes so aparecem no app quando o modo dev estiver rodando com sucesso.

Se quiser um exe puro sem Node/Rust, sera preciso gerar uma build Windows separada.
